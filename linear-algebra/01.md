# linear-algebra 01

## dot 点乘

```c#
vec a = vec(x1, y1, z1, ...);
vec b = vec(x2, y2, z2, ...);
dot(a, b) === len(a) * len(b) * cos(a, b)
          === x1 * x2 + y1 * y2 + z1 * z2 + ...;
```

点乘满足交换律、结合律、分配律

## project 投影

```c#
vec a = vec(x1, y1, z1, ...);
vec b = vec(x2, y2, z2, ...);
float proB = dot(a, b) / len(a);
```
